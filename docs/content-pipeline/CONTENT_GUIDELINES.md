# Content Guidelines

## Content folders
- `content/cards`
- `content/heroes`
- `content/companions`
- `content/campaigns`
- `content/cosmetics`

## Principles
1. Keep content data-driven and versioned.
2. Avoid embedding gameplay numbers directly in UI/server code.
3. Validate all content through schema checks before release.
4. Make color/class restrictions explicit on each card/ability.
